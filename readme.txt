TATA CARA PENGGUNAAN PROGRAM mytools
Sintaks utama : mytools [nama_file]
Sintaks diatas bertujuan untuk outputkan file apapun ke dalam ekstensi .txt
-t : Flag untuk outputkan file dengan ekstensi file secara spesifik
Sintaks penggunaan : mytools [nama_file] -t [nama_ekstensi]. Nama extensi yang diperbolehkan adalah json(*.json) dan text(*.txt)
-o : Flag untuk outputkan file dengan nama file dan pada folder secara spesifik
Sintaks penggunaan : mytools [nama_file] -o [folder_path]
Bila ingin mengoutputkan file dengan nama folder, file dan extensi file secara spesifik
Sintaks penggunaan : mytools [nama_file] -t [nama_tipe] -o [folder_path]
-h : Flag untuk menampilkan bantuan dan sintaks pada program ini
Selain daripada sintaks diatas, maka program akan error
